# Docker Container Monitor



Проект для автоматического мониторинга Docker-контейнеров с помощью bash-скриптов.



## Функциональность



- Автоматическое создание отчетов о запущенных контейнерах

- Логирование с timestamp в имени файла

- Сохранение результатов в текстовые файлы



## Скрипты



\- `docker\_log.ps1` - основной скрипт для создания отчетов

\- `demo\_script.ps1` - демонстрационная версия с обработкой ошибок
